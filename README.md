# University LMS Portal

A complete Learning Management System (LMS) portal for university students and teachers.


## Author
Muhammad Abu Bakar

## License
This project is for educational purposes.

### Student Dashboard
- Course registration system
- Course schedule view
- Student advisor portal
- Assignment submission tracking
- Grades and assessments

### Teacher Dashboard
- Course management
- Assignment creation
- Quiz creation with deadlines
- Student grade management
- Course materials upload

## Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage for authentication
- Font Awesome icons

## How to Run
1. Clone the repository
2. Open `index.html` in your browser for student portal
3. Open `Teacher Portal.html` for teacher dashboard
