A complete Learning Management System (LMS) portal for university students, teachers, and admins.

### Student Dashboard
- Course registration system
- Course schedule view
- Student advisor portal
- Profile settings (update name, password, profile pic)
- Assignment submission tracking
- Grades and assessments

### Teacher Dashboard
- Course management
- Assignment creation
- Quiz creation with deadlines
- Student grade management
- Course materials upload
- Profile settings (update credentials)

### Admin Dashboard
- Add new users (Students, Teachers, Admins)
- Create courses/sections with teacher, day, time, and room
- Manage system-wide settings

## Technologies Used
- React.js (Frontend)
- Node.js + Express (Backend API)
- MySQL (Database)
- JWT Authentication
- Webpack (Build tool)
- Font Awesome icons

## How to Run Locally
1. Clone the repository
2. Run `npm install` in the root directory
3. Run `npm install` in the `backend/` directory
4. Set up your MySQL database using `backend/database.sql`
5. Configure `backend/.env` with your database credentials
6. Run `npm start` in the `backend/` folder to start the API server
7. Run `npm start` in the root folder to start the React dev server
